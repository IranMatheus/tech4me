Exemplos de Código HTML Úteis

1. Título e Parágrafo

Este snippet demonstra como criar um título e um parágrafo simples em HTML.

<h1>Olá, Mundo!</h1>
<p>Este é um parágrafo de exemplo.</p>

2. Link para Outra Página

Este snippet mostra como criar um link para outra página da web.

<a href="pagina2.html">Visite a Página 2</a>

3. Lista Não Ordenada

Este snippet ilustra como criar uma lista não ordenada (com marcadores).

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
